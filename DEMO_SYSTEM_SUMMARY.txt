   ```go

   demoService := services.NewDemoResetService(db)
   demoService.StartScheduler()
   ```

2. Load migration 025 in database

3. Optionally schedule cron job:
   ```bash
   0 0 1 * * /path/to/scripts/reset-demo-data.sh
   ```

================================================================================
AUTO-RESET MECHANISM
================================================================================

HOW IT WORKS:

1. App Startup:
   - DemoResetService initializes
   - Checks if reset is needed
   - Runs first reset if required
   - Starts 30-day scheduler

2. Every 30 Days:
   - Timer triggers reset
   - Deletes all demo data (18 tables)
   - Reloads fresh demo data
   - Logs reset completion

3. Data Safety:
   - Uses database transactions
   - FK_CHECKS=0 during delete
   - INSERT IGNORE prevents duplicates
   - Error handling prevents partial resets

4. Monitoring:
   - Logs all reset events
   - Timestamps for audit trail
   - Error messages for troubleshooting
